Real Time Operating System (RTOS) Project

4-member Group Project
Project Period: Jan 2022 - Apr 2022
Programming Tools: C, LPCXpresso v6.1.4
Hardwares: FRDM-KL25z ARM-Cortex M0 microcontroller, ESP32, ultrasonic sensor, AA-batteries

This project is about developing a real-time operating system robotic car that can be remotely controlled using mobile phone. The robotic car we come out with have two modes, the first mode is allowed remote-controlling, while the second mode only supports self-driving.

In this project, we completed in a group of 4. I used the knowledge learned in the lectures such as GPIO, interrupts, semaphore, mutex, message queue, and UART to write the code for the robotic car. I also helped the team to set up and callibrate the ultrasonic sensor and speaker.

The most challenging part of this project is the remote connection between the microcontroller and the mobile phone. We are using ESP32 which helps to connect the microcontroller FRDM-KL25z and the phone device through WiFi. We always lost the connection due to unstable WiFi. After numerous trials, we realize that it is because the battery supply to the ESP32 is draining fast and when the electricity becomes lower, the connection is not stable. Hence, we need to prepare a lot of batteries for backup.

Implement Searching Algorithms in AI

Individual Project
Project Period: Feb 2022 - Apr 2022
Programming Tools: Python, Pycharm

This project consists of three tasks. Task 1 is to implement BFS, DFS, Dijkstra and A* algorithms for the computer to find the path through the maze which contains a lot of obstacles/walls. Task 2 is to implement local search and constraint-satisfaction problem (CSP) to solve N-piece problem (a problem that is similar to N-queen except that the queen is replaced with any possible chess pieces). Task 3 requires us to implement the adversarial search algorithm to design a mini-chess AI that is good enough to take rational movement and have high win rate.

This project is a self-development project, and it takes 3 weeks to complete each task. The difficulty of the project is increasing as I move on, and I greatly appreciate that I can apply the AI search algorithms learned in the lecture to a problem.

The hardest part of this problem is to model the given situation in a way such that computer can understand and analyze the situation. Furthermore, the efficiency of the code is also a big challenge. Throughout this project, I have better understand the use case of different searching algorithms used in AI.
